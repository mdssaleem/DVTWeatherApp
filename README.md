# DVTWeatherApp


A clean, modern iOS weather application that displays **5-day weather forecasts** for your current location or a selected city. Built with **SwiftUI** and follows **MVVM architecture** for maintainable and scalable code.  

---

## Features

- **5-Day Weather Forecast:** Shows daily weather with temperature, icon, and day name.  
- **Detailed Weather View:** Tap on a card to see full weather info including humidity, pressure, min/max temperatures, and description.  
- **Dynamic Background:** Full-screen background image changes based on weather conditions.  
- **Responsive UI:** Cards have shadows, rounded corners, and proper spacing for a clean look.  
- **Custom Fonts:** Poppins font applied consistently across the app.  
- **Interactive Elements:** Tapping a card shows a modal popup with detailed weather info.
